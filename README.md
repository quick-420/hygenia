# Hygenia - Medical Appointment Booking System

Hygenia is a modern web application that helps patients find and book appointments with doctors in their area. The platform provides a clean, user-friendly interface for managing medical appointments and service requests.

## Features

- Modern, responsive UI built with Material-UI
- Appointment booking system
- Service request management
- MongoDB database integration
- RESTful API backend

## Prerequisites

- Node.js (v14 or higher)
- MongoDB (v4.4 or higher)
- npm or yarn package manager

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd hygenia
```

2. Install backend dependencies:
```bash
npm install
```

3. Install frontend dependencies:
```bash
cd client
npm install
```

4. Create a `.env` file in the root directory with the following variables:
```
MONGODB_URI=mongodb://localhost:27017/hygenia
PORT=5000
```

## Running the Application

1. Start the backend server:
```bash
npm run dev
```

2. In a new terminal, start the frontend development server:
```bash
cd client
npm start
```

The application will be available at:
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000
